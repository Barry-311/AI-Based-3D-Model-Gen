## MVP功能：
- 基础网页搭建（网址、页面排版、基本交互）  
- 文本 / 图片输入（多模态支持）  
- 调用 API 生成单个 3D 模型  
- 模型下载功能（支持 `.glb / .obj / .fbx` 格式）
